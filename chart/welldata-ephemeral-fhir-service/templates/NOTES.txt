  http{{ if $.Values.fhirServer.ingress.tls }}s{{ end }}://{{ $host.host }}{{ .path }}
  {{- end }}
{{- end }}
{{- else }}
The FHIR server is available within the cluster at:
  http://{{ include "welldata.fhirServer.name" . }}:{{ .Values.fhirServer.service.port }}

To access it locally, run:
  kubectl port-forward svc/{{ include "welldata.fhirServer.name" . }} 8080:{{ .Values.fhirServer.service.port }}

Then access: http://localhost:8080/fhir/metadata
{{- end }}
